1. Нужно для каждой программы - серверной и клиентской установить необходимые 
библиотеки перейдя в соответсвующую корневую папку программы и выполнив команду
в терминале: 
npm install
2. В файле .env нужно вставить свой ключ для работы с запосами на чат gpt OPENAI_API_KEY.
3. Запустить серверную часть можно командой:
npm run dev
4. Запустить клиентскую часть можно командой:
npm start
